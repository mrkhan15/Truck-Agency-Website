# Truck Agency One-Page Website

A sleek and fully responsive one-page truck agency website created to showcase my front-end web development skills, including responsive layout design, clean UI implementation, and a bilingual user experience.

## Overview

This project is a modern landing page concept for a truck agency. I created it as a portfolio/demo piece to showcase my web development skills through a minimal visual style, strong typography, separate HTML/CSS/JavaScript structure, and an English/Espanol language toggle.

## Highlights

- Fully responsive one-page website
- Clean and minimal UI design
- Separate `HTML`, `CSS`, and `JavaScript` files
- Mobile-friendly layout and navigation
- English / Espanol language toggle
- Smooth scroll and section-based navigation
- Lightweight reveal animations for content sections

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
